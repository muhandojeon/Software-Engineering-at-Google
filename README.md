## 진행 방식 🌌

* 스터디 세션 전에 주차에 해당하는 장을 읽고 정리한 내용을 PR로 올려요
* 스터디 세션에서 각자 아래 주제에 대해 공유해요
  + 중요하다고 생각한 내용
  + 경험적인 내용
  + 이해가 되지 않는 내용
* 각 장마다 정해진 서기가 공유 내용을 기록하고 저장소에 올려요
